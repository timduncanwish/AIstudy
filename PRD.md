# PRD.md — AI助学产品需求文档

## 一、产品概述

**产品名称：** AI助学

**目标：** 通过AI帮助家长辅导小学3-6年级学生的语文和英语，减轻家长负担，提升孩子学习效果。

**平台：** 微信小程序（家长和孩子都能随时使用）

---

## 二、用户画像

### 核心用户：小学生（3-6年级）
- 年龄：8-12岁
- 特点：注意力有限，需要趣味性和即时反馈
- 需求：不理解的知识点需要耐心解释，做错的题目需要明白为什么错

### 决策用户：家长
- 年龄：30-45岁
- 痛点：工作忙没时间辅导、自己也不确定答案对不对、辅导时容易情绪化
- 需求：了解孩子学了什么、哪些地方薄弱、AI辅导质量可靠

---

## 三、功能模块

### 模块1：AI对话辅导（一期 — 已开发基础版本）

**功能描述：** 孩子通过文字对话与AI老师交流，获得语文或英语的个性化辅导。

**详细需求：**

| 功能点 | 说明 |
|--------|------|
| 科目选择 | 首页选择"语文辅导"或"英语辅导"进入对应对话 |
| 年级适配 | 3-6年级切换，AI自动调整语言难度和知识范围 |
| 多轮对话 | 支持上下文连续对话，AI能记住前文 |
| 对话历史 | 本地保存对话记录，可回看 |
| 引导式教学 | AI不直接给答案，通过提问引导孩子思考 |
| 错误纠正 | 孩子答错时温和指出并解释原因 |

**语文辅导场景：**
- 课文理解："这段话是什么意思？"
- 字词学习：""绚烂"是什么意思？怎么用？"
- 作文指导："怎么写一篇关于春天的作文？"
- 阅读理解训练：给出短文+提问

**英语辅导场景：**
- 单词学习："apple是什么意思？"
- 语法讲解："什么时候用a，什么时候用an？"
- 情景对话：模拟日常英语对话
- 翻译辅助：中英文互译

### 模块2：字词句课前预习（二期新增）

**功能描述：** 按各年级上下学期教材提供语文和英语的字、词、句预习，帮助孩子在上课前完成基础认读、理解和简单运用。

**教材标准：**
- 教材来源优先使用官方公开电子教材入口：国家中小学智慧教育平台、人教社中小学教材电子版等。
- 教材库覆盖1-9年级上下册，产品首期开放3-6年级，后续可扩展到初中衔接。
- 语文以统编版教材为主，按教材课后或书后一类字、二类字、生字表、词语表建立预习清单。
- 英语以学校常用教材版本为准，首期默认PEP/人教版，按教材后面的词汇表建立单词和重点句型清单。
- 不在小程序内分发完整教材PDF或整页扫描图，只保存学习所需的结构化字词句数据、教材版本、年级、册次、单元、课文/模块、页码引用和来源校验记录。

**详细需求：**

| 功能点 | 说明 |
|--------|------|
| 年级学期选择 | 支持1-9年级、上/下学期、教材版本选择；当前产品默认展示3-6年级 |
| 语文预习 | 按课文展示一类字、二类字、词语、重点句；一类字要求会认、会写、会组词，二类字要求会认、理解 |
| 英语预习 | 按单元展示词汇表单词、短语、重点句型；提供发音、中文解释、例句和简单情景用法 |
| 字词卡片 | 每个字/词包含拼音或音标、释义、例句、易错点、书写/拼读提醒 |
| 句子练习 | 语文做仿句和口头表达，英语做跟读、替换造句和中英互译 |
| AI讲解 | AI根据年级用简单语言解释，不直接灌输答案，先引导孩子观察再总结 |
| 预习进度 | 记录每课/每单元的认读、理解、运用完成情况 |
| 家长摘要 | 向家长展示本周已预习课文/单元、薄弱字词、建议复习项 |

**语文预习流程：**
1. 选择年级、学期、课文。
2. 系统展示本课一类字、二类字和重点词句。
3. 孩子完成认读、组词、造句和易错字辨析。
4. AI用鼓励式反馈指出需要再练的字词。

**英语预习流程：**
1. 选择年级、学期、单元。
2. 系统展示本单元词汇表和重点句型。
3. 孩子完成听读、看图选词、拼写和替换造句。
4. AI用简单英文加中文解释，帮助孩子理解使用场景。

**验收标准：**
- 任一3-6年级语文课文可打开对应一类字、二类字和重点词句预习清单。
- 任一3-6年级英语单元可打开对应词汇表和重点句型预习清单。
- 每个预习项都有年级、学期、教材版本、册次、单元/课文和来源记录。
- AI反馈符合“先肯定、再提示、最后给练习建议”的正面激励原则。

**字表/词汇表采集标准（数据落库依据）：**

| 科目 | 版本 | 采集对象 | 对应教学要求 | 取自教材位置 |
|------|------|----------|--------------|--------------|
| 语文 | 统编版（部编版） | **生字表（二）= 会写字 = 一类字** | 会认、会写、会组词、会用 | 每册书后“写字表” |
| 语文 | 统编版（部编版） | **生字表（一）= 会认字 = 二类字** | 会认、理解 | 每册书后“识字表” |
| 英语 | PEP人教版 | **单词表（黑体词为重点词）+ 短语 + 重点句型** | 听读、理解、拼写、替换造句 | 每单元词汇表 / 书后 Word List |

- 每条字/词记录字段：年级、学期（上/下册）、教材版本、单元/课文、字或词、拼音/音标、释义、例句、来源URL、`source_status`（`待人工复核` / `已复核`）。
- 数据先以官方公开电子教材为来源采集，落库后统一标 `待人工复核`，人工核对无误后改 `已复核` 才视为可正式发布。
- 只保存结构化字词句数据与页码/单元引用，**不在产品内存储或分发整页教材扫描件、PDF**。

**1-9 年级教材数据来源（官方公开入口）：**

| 范围 | 语文（统编版）来源 | 英语（PEP人教版）来源 |
|------|--------------------|------------------------|
| 1-9 年级全册电子教材 | 国家中小学智慧教育平台 https://basic.smartedu.cn/ | 国家中小学智慧教育平台 https://basic.smartedu.cn/ |
| 人教社电子课本 | https://bp.pep.com.cn/jc/ | https://jc.pep.com.cn/ |
| 字表/词汇表校验 | 各册书后“识字表/写字表” | 各册书后 Word List / 单元词汇表 |

- 覆盖范围：1-9 年级上下册共 36 册（语文 18 + 英语 18）；产品首期开放 3-6 年级，1-2、7-9 年级数据随采随上。
- 采集进度与每册复核状态见 `backend/app/data/textbooks/INDEX.md`（数据落库时同步维护）。

### 模块3：作业批改+讲解（三期）

**功能描述：** 孩子拍照上传作业，AI识别文字/手写内容，批改并逐题讲解。

| 功能点 | 说明 |
|--------|------|
| 拍照上传 | 调用微信相机拍照或从相册选择 |
| OCR识别 | 识别手写和印刷体文字 |
| AI批改 | 逐题判断对错，标注错误位置 |
| 错题讲解 | 对每道错题生成详细讲解 |
| 错题本 | 错题自动归档，按科目/知识点分类 |
| 错题复习 | 按遗忘曲线推送复习提醒 |

---

### 模块4：生字/单词闯关（四期）

**功能描述：** 游戏化的字词学习，按教材分级，通过闯关模式巩固记忆。

| 功能点 | 说明 |
|--------|------|
| 年级词库 | 复用字词句预习教材库，按语文一类字/二类字和英语词汇表分级 |
| 闯关模式 | 认读→拼写→造句→应用，四级递进 |
| 每日任务 | 每天推送10个生字/单词 |
| 积分徽章 | 完成关卡获得积分和徽章激励 |
| 复习提醒 | 基于遗忘曲线的智能复习推送 |
| 学习报告 | 每周生成字词掌握情况报告 |

---

### 模块5：家长端（五期）

| 功能点 | 说明 |
|--------|------|
| 微信登录 | 家长微信一键登录 |
| 学生档案 | 添加多个孩子，设置年级 |
| 学习报告 | 每日/每周学习时长、科目分布、薄弱项分析 |
| 对话查看 | 查看孩子和AI的对话记录 |
| 学习提醒 | 设置每日学习提醒时间 |

## 五、非功能需求

| 需求 | 目标 |
|------|------|
| AI响应时间 | < 3秒（首字） |
| 可用性 | 99.5% |
| 并发 | 初期支持100并发 |
| 隐私 | 儿童数据加密存储，不对外共享 |
| 合规 | 符合《儿童个人信息网络保护规定》 |
| 教材版权 | 只使用官方公开来源做数据校验，不在产品内分发完整教材内容 |
| 教材准确性 | 字词句数据需保留来源、版本、年级、册次和人工复核状态 |
