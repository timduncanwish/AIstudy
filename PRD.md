# PRD.md — AI助学产品需求文档

## 一、产品概述

**产品名称：** AI助学

**目标：** 通过AI帮助家长辅导小学3-6年级学生的语文和英语，减轻家长负担，提升孩子学习效果。

**平台：** 微信小程序（家长和孩子都能随时使用）

---

## 二、用户画像

### 核心用户：小学生（3-6年级）
- 年龄：8-12岁
- 特点：注意力有限，需要趣味性和即时反馈
- 需求：不理解的知识点需要耐心解释，做错的题目需要明白为什么错

### 决策用户：家长
- 年龄：30-45岁
- 痛点：工作忙没时间辅导、自己也不确定答案对不对、辅导时容易情绪化
- 需求：了解孩子学了什么、哪些地方薄弱、AI辅导质量可靠

---

## 三、功能模块

### 模块1：AI对话辅导（一期 — 已开发基础版本）

**功能描述：** 孩子通过文字对话与AI老师交流，获得语文或英语的个性化辅导。

**详细需求：**

| 功能点 | 说明 |
|--------|------|
| 科目选择 | 首页选择"语文辅导"或"英语辅导"进入对应对话 |
| 年级适配 | 3-6年级切换，AI自动调整语言难度和知识范围 |
| 多轮对话 | 支持上下文连续对话，AI能记住前文 |
| 对话历史 | 本地保存对话记录，可回看 |
| 引导式教学 | AI不直接给答案，通过提问引导孩子思考 |
| 错误纠正 | 孩子答错时温和指出并解释原因 |

**语文辅导场景：**
- 课文理解："这段话是什么意思？"
- 字词学习：""绚烂"是什么意思？怎么用？"
- 作文指导："怎么写一篇关于春天的作文？"
- 阅读理解训练：给出短文+提问

**英语辅导场景：**
- 单词学习："apple是什么意思？"
- 语法讲解："什么时候用a，什么时候用an？"
- 情景对话：模拟日常英语对话
- 翻译辅助：中英文互译

---

### 模块2：作业批改+讲解（二期）

**功能描述：** 孩子拍照上传作业，AI识别文字/手写内容，批改并逐题讲解。

| 功能点 | 说明 |
|--------|------|
| 拍照上传 | 调用微信相机拍照或从相册选择 |
| OCR识别 | 识别手写和印刷体文字 |
| AI批改 | 逐题判断对错，标注错误位置 |
| 错题讲解 | 对每道错题生成详细讲解 |
| 错题本 | 错题自动归档，按科目/知识点分类 |
| 错题复习 | 按遗忘曲线推送复习提醒 |

---

### 模块3：生字/单词闯关（三期）

**功能描述：** 游戏化的字词学习，按教材分级，通过闯关模式巩固记忆。

| 功能点 | 说明 |
|--------|------|
| 年级词库 | 按部编版语文教材和PEP英语教材分级 |
| 闯关模式 | 认读→拼写→造句→应用，四级递进 |
| 每日任务 | 每天推送10个生字/单词 |
| 积分徽章 | 完成关卡获得积分和徽章激励 |
| 复习提醒 | 基于遗忘曲线的智能复习推送 |
| 学习报告 | 每周生成字词掌握情况报告 |

---

### 模块4：家长端（四期）

| 功能点 | 说明 |
|--------|------|
| 微信登录 | 家长微信一键登录 |
| 学生档案 | 添加多个孩子，设置年级 |
| 学习报告 | 每日/每周学习时长、科目分布、薄弱项分析 |
| 对话查看 | 查看孩子和AI的对话记录 |
| 学习提醒 | 设置每日学习提醒时间 |
