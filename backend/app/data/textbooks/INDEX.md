# 教材预习数据索引（字词句课前预习）

本目录存放「字词句课前预习」的真实教材数据，按学期（上/下册）组织。
- 语文：统编版（部编版）。一类字 = 写字表（会写）；二类字 = 识字表（会认）。
- 英语：PEP人教版。按单元词汇表（黑体词为重点）+ 短语 + 重点句型。
- 与 `word_banks/`（字词闯关游戏，含干扰项）分离，互不影响。

## 数据来源（官方公开入口）
- 国家中小学智慧教育平台：https://basic.smartedu.cn/
- 人教社电子课本：语文 https://bp.pep.com.cn/jc/ ，英语 https://jc.pep.com.cn/

## 复核状态说明
每个单元含 `source_status` 字段：
- `待人工复核`：已按教材整理，尚未人工逐字核对，**不得正式发布**。
- `已复核`：人工核对字表/词汇表无误，可发布。

> 当前所有单元均为「待人工复核」。请按官方教材逐册核对一类字/二类字与单词表的**完整性**与**逐课归属**后改为「已复核」。
