# CLAUDE.md — AI助学项目

## 项目价值观

1. **孩子第一** — 所有设计决策以3-6年级小学生的认知水平为核心，语言简洁、互动友好
2. **辅助家长** — AI是家长的助教，不是替代品。让家长了解孩子的学习情况，减轻辅导压力
3. **分年级精准** — 3年级和6年级的知识体系、表达方式完全不同，内容必须年级适配
4. **正面激励** — 不批评、不打分，用鼓励和引导的方式帮助孩子建立学习信心
5. **隐私保护** — 儿童数据严格保护，不对外泄露任何学习记录和个人信息

## 产品定位

**一句话描述：** AI辅助小学3-6年级学生打好语文和英语基础的微信小程序。

**目标用户：**
- 主要用户：小学3-6年级学生
- 决策者/付费者：学生家长
- 核心场景：课后辅导、作业答疑、字词积累

## 技术栈

| 层级 | 技术 | 版本/说明 |
|------|------|-----------|
| 前端 | uni-app (Vue 3 + TypeScript) | 微信小程序 |
| 后端 | Python FastAPI | 异步API服务 |
| AI | 智谱AI (GLM-4-flash) | OpenAI兼容接口 |
| 数据库 | SQLite (开发) / MySQL (生产) | |
| OCR | 百度OCR / 腾讯OCR | 作业批改用（二期） |

## 开发约定

- 前端用 rpx 单位，适配不同屏幕
- 后端 API 统一前缀，返回 JSON
- 智谱AI调用通过 OpenAI SDK 兼容接口
- `.env` 文件不提交到版本控制
- commit 信息用中文

## AI提示词原则

- 语文提示词用中文，按年级调整用语难度
- 英语提示词用英文，要求AI用简单英文+中文解释
- AI不直接给答案，引导式教学
- 错误纠正要温和，先肯定再指出改进方向
